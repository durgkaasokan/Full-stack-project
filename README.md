
# FULL STACK DEVELOPMENT

A full-stack web application featuring a decoupled architecture with a robust backend API and a responsive user interface.

---

## 🛠️ Tech Stack
* **Frontend:** React.js / Tailwind CSS / Context API
* **Backend:** Node.js (Express.js) / JWT Authentication
* **Database:** MongoDB / PostgreSQL
* **Deployment:** Vercel (Frontend), Render (Backend)
  
---

## 📂 Project Structure

```text
├── frontend/     # Client-side UI and routing
├── backend/      # API server, models, and controllers
└── README.md

```

---

## 🔑 Key API Endpoints

* `POST /api/auth/register` - User registration
* `POST /api/auth/login` - User authentication (returns JWT)
* `GET /api/data` - Fetch resources
* `POST /api/data` - Create resource (Protected)
